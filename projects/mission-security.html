Mission control security systems

NASA Johnson Space Center, 2025

Problem

NASA's Mission Control Center systems require rigorous security review for any infrastructure change. Firewall rules, network access controls, Linux authentication changes all go through a formal review process with multiple control boards. A planned Critical Design Review was estimated at 500+ labor hours across several organizations.

Approach

Managed the full change request workflow in ServiceNow. Gathered technical requirements from stakeholders, validated network configurations using nslookup and hostname mapping, and presented firewall and security changes to cross-functional control boards.

For the CDR: did a technical feasibility analysis and presented it to IT security groups. Showed that existing security controls and precedent from previous implementations already covered the requirements. The result was eliminating the design phase entirely, saving those 500+ labor hours.

Tech stack

ServiceNow Linux nslookup Network security Firewall rules

Status

Changes implemented. CDR design phase eliminated. Saved 500+ estimated labor hours.
